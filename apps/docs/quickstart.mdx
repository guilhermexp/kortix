## 1. Preparar o ambiente

1. Acesse `http://localhost:3000`, entre com o usuário administrador e vá em **Configurações → API Keys** para gerar um token.
2. No seu terminal, exporte o token e defina a URL do backend (ajuste conforme o domínio que você publicar):

```bash
export SUPERMEMORY_API_KEY="SEU_TOKEN_GERADO"
export SUPERMEMORY_API_URL="http://localhost:4000"
```

3. Opcional: crie variáveis equivalentes em arquivos `.env` para projetos locais.

## 2. Adicionar documentos e memórias

A API trabalha com documentos (`/v3/documents`) que viram memórias após o pipeline de ingestão. Durante o desenvolvimento a resposta já retorna o status do job.

### cURL

```bash
curl -X POST "$SUPERMEMORY_API_URL/v3/documents" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Notas de Quantum Computing",
    "type": "text",
    "projectId": "default",
    "content": "Quantum computing representa uma mudança de paradigma...",
    "metadata": {
      "tags": ["pesquisa", "aulas"],
      "source": "notion"
    }
  }'
```

Resposta de exemplo:

```json
{
  "documentId": "doc_01hxyz...",
  "status": "queued"
}
```

### Node.js / TypeScript

```ts
import "dotenv/config"

const API_URL = process.env.SUPERMEMORY_API_URL ?? "http://localhost:4000"
const API_KEY = process.env.SUPERMEMORY_API_KEY!

async function createDocument() {
  const response = await fetch(`${API_URL}/v3/documents`, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      name: "Notas de Quantum Computing",
      type: "text",
      projectId: "default",
      content: "Quantum computing representa uma mudança de paradigma...",
      metadata: {
        tags: ["pesquisa", "aulas"],
        source: "notion",
      },
    }),
  })

  if (!response.ok) {
    throw new Error(`Falha ao criar documento: ${response.status}`)
  }

  const payload = await response.json()
  console.log(payload)
}

createDocument().catch(console.error)
```

### Python

```python
import os
import requests

API_URL = os.getenv("SUPERMEMORY_API_URL", "http://localhost:4000")
API_KEY = os.environ["SUPERMEMORY_API_KEY"]

payload = {
    "name": "Notas de Quantum Computing",
    "type": "text",
    "projectId": "default",
    "content": "Quantum computing representa uma mudança de paradigma...",
    "metadata": {
        "tags": ["pesquisa", "aulas"],
        "source": "notion"
    }
}

response = requests.post(
    f"{API_URL}/v3/documents",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    },
    json=payload,
    timeout=30
)

response.raise_for_status()
print(response.json())
```

## 3. Consultar memórias

Depois da ingestão o job cria chunks e memórias associadas. Use `/v3/search` para recuperar contexto relevante.

```bash
curl -X POST "$SUPERMEMORY_API_URL/v3/search" \
  -H "Authorization: Bearer $SUPERMEMORY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "projectId": "default",
    "query": "Quais são os usos principais de quantum computing?",
    "limit": 3
  }'
```

Resposta resumida:

```json
{
  "results": [
    {
      "documentId": "doc_01hxyz...",
      "chunkId": "chk_01h123...",
      "score": 0.82,
      "content": "Quantum computing é aplicado em criptografia...",
      "metadata": {
        "tags": ["pesquisa", "aulas"],
        "source": "notion"
      }
    }
  ],
  "total": 1
}
```
